# agentic-browser

Browser automation for AI agents. Control Chrome via the DevTools Protocol from your AI coding assistant, CLI, or TypeScript code.

## Three Ways to Use It

### MCP Server (Recommended)

Configure `agentic-browser mcp` as an MCP server in your AI tool. No shell commands needed — tools appear directly in your assistant.

```json
{
  "mcpServers": {
    "agentic-browser": {
      "command": "npx",
      "args": ["agentic-browser", "mcp"]
    }
  }
}
```

### CLI

Shell out to the `agentic-browser` CLI for stateful browser automation with auto-restart and retry:

```bash
agentic-browser agent start
agentic-browser agent navigate https://example.com
agentic-browser agent elements
agentic-browser agent content --mode summary
agentic-browser agent stop
```

### Programmatic API

Import `AgenticBrowserCore` directly in your TypeScript code:

```ts
import { createAgenticBrowserCore } from "agentic-browser";

const core = createAgenticBrowserCore();
const session = await core.startSession();
await core.runCommand({
  sessionId: session.sessionId,
  commandId: "nav-1",
  type: "navigate",
  payload: { url: "https://example.com" },
});
```
