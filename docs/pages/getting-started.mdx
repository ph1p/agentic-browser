# Getting Started

## Prerequisites

- **Node.js 20+**
- **Chrome** installed (auto-discovered, or set `AGENTIC_BROWSER_CHROME_EXECUTABLE_PATH`)

## Install

```bash
npm install agentic-browser -g
```

## Quick Start: CLI

Start a browser session, navigate to a page, discover elements, and read content:

```bash
# 1. Start Chrome
agentic-browser agent start

# 2. Navigate
agentic-browser agent run navigate '{"url":"https://example.com"}'

# 3. Discover what's on the page
agentic-browser agent elements

# 4. Click something using a discovered selector
agentic-browser agent run interact '{"action":"click","selector":"a[href=\"/more\"]"}'

# 5. Read page content
agentic-browser agent content --mode text

# 6. Done
agentic-browser agent stop
```

Every command prints exactly one JSON line to stdout.

### Use your existing Chrome

Want your logged-in sessions, cookies, and extensions? Quit Chrome, relaunch it with remote debugging, and connect:

```bash
# 1. Quit Chrome, then relaunch with debugging enabled
/Applications/Google\ Chrome.app/Contents/MacOS/Google\ Chrome --remote-debugging-port=9222
# (Linux: google-chrome --remote-debugging-port=9222)

# 2. Connect agentic-browser to it
agentic-browser agent start --cdp-url http://127.0.0.1:9222
```

Chrome opens normally with all your tabs and sessions. Stopping the agent will **not** close your Chrome.

Alternatively, launch a new Chrome with your default profile (requires quitting Chrome first):

```bash
agentic-browser agent start --user-profile default
```

## Quick Start: MCP Server

If your AI tool supports [MCP](https://modelcontextprotocol.io), configure the stdio server instead of using CLI commands.

### Automatic Setup

```bash
npx agentic-browser setup
```

Detects Claude Code, Cursor, and writes the config for you.

### Manual: Claude Code

Add to `.mcp.json` in your project root:

```json
{
  "mcpServers": {
    "agentic-browser": {
      "command": "npx",
      "args": ["agentic-browser", "mcp"]
    }
  }
}
```

### Manual: Cursor

Add to `.cursor/mcp.json`:

```json
{
  "mcpServers": {
    "agentic-browser": {
      "command": "npx",
      "args": ["agentic-browser", "mcp"]
    }
  }
}
```

Once configured, you'll see 8 browser tools available in your assistant:
`browser_start_session`, `browser_navigate`, `browser_interact`, `browser_get_content`, `browser_get_elements`, `browser_search_memory`, `browser_dismiss_cookies`, `browser_stop_session`.

## Quick Start: Programmatic API

```ts
import { createAgenticBrowserCore } from "agentic-browser";

const core = createAgenticBrowserCore();
const session = await core.startSession();

// Navigate
await core.runCommand({
  sessionId: session.sessionId,
  commandId: "nav-1",
  type: "navigate",
  payload: { url: "https://example.com" },
});

// Discover elements
const elements = await core.getInteractiveElements({
  sessionId: session.sessionId,
  visibleOnly: true,
  limit: 30,
});

console.log(elements.elements.map((e) => `${e.role}: ${e.text} → ${e.selector}`));

// Clean up
await core.stopSession(session.sessionId);
```

## Environment Variables

| Variable                             | Default            | Description                                                         |
| ------------------------------------ | ------------------ | ------------------------------------------------------------------- |
| `AGENTIC_BROWSER_LOG_DIR`            | `.agentic-browser` | Base directory for sessions, memory, events                         |
| `AGENTIC_BROWSER_CHROME_EXECUTABLE_PATH` | _(auto-discover)_  | Explicit path to Chrome executable                              |
| `AGENTIC_BROWSER_CHROME_PATH`            | _(legacy alias)_   | Backward-compatible alias for Chrome executable path            |
| `AGENTIC_BROWSER_CDP_URL`            |                    | Connect to an already-running Chrome (e.g. `http://127.0.0.1:9222`) |
| `AGENTIC_BROWSER_USER_PROFILE`       |                    | Use real Chrome profile (`default`, `true`, or an absolute path)    |
| `AGENTIC_BROWSER_HEADLESS`           |                    | Run Chrome in headless mode (`true` or `1`)                         |
| `AGENTIC_BROWSER_USER_AGENT`         |                    | Override the browser user-agent string                              |
| `AGENTIC_BROWSER_COMMAND_TIMEOUT_MS` | `2000`             | Command execution timeout in milliseconds                           |
| `AGENTIC_BROWSER_HOST`               | `127.0.0.1`        | WebSocket server host                                               |
| `AGENTIC_BROWSER_WS_PORT`            | `43111`            | WebSocket server port                                               |
